# Figma is a link, not an integration

A Finding or Review Session may carry a Design Reference — a pasted Figma URL, optional at both levels. We deliberately excluded rendering the design inside nitpick (side-by-side, overlay, pixel-diff): it requires Figma auth, frame selection, and resolution matching, and turns a review-capture tool into a design-comparison product. If comparison rendering is ever proposed, it is a new product decision, not a feature request.
